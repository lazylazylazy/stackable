# Stackable

Stackable is a an Ruby API wrapper for the unoffical Stack Over Flow jobs API.

## Installation

Add this line to your application's Gemfile:

    gem 'stackable'

And then execute:

    $ bundle

Or install it yourself via rubygems:

    $ gem install stackable
